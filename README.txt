Justificador Program(R) version 1.0 30/01/2018

GENERAL USAGE NOTES
-------------------

-Justificador no soporta Windows NT Server or Windows 2000 Servers.

-Justificador es un programa que permite gestionar las faltas, retrasos, asistencias de alumnos en un centro de enseñanza.

---------------------------------------------------

REQUISITOS DE INSTALACIÓN:
--------------------------

-Se recomienda su instalación en Windows Vista o superior, Ubuntu 13.0 o superior.

-Se recomienda tener al menos 1GB de RAM.

-Se recomienda tener al menos 10GB de espacio en el disco duro.
